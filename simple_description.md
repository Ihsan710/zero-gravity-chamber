# Zero-Gravity Chamber 🌌

**Zero-Gravity Chamber** is an interactive 3D web experience where you control the laws of physics. 

**What it does:**
*   **Refy Gravity:** Click a button to reverse gravity and watch neon objects float into space.
*   **Cinematic Experience:** The camera flips upside down, and the background shifts to a deep starfield.
*   **Real-Time Physics:** Objects collide, bounce, and rotate realistically using the Cannon-es physics engine.
*   **Sci-Fi Vibes:** Features procedural sound effects, bloom lighting, and a "Matrix-style" time control slider.

Built with **Three.js** and **JavaScript**, this project runs directly in your browser to demonstrate the power of modern 3D web graphics. No plugins required—just click and float.
